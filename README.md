# PFT Telecom – Python e Inteligência Artificial

Este repositório reúne os códigos desenvolvidos durante as atividades da fase inicial do **PFT Telecom**, programa promovido pelo xGMobile, no primeiro semestre de 2026.

As disciplinas de **Python** e **Inteligência Artificial** abordam conceitos fundamentais para a compreensão, implementação e avaliação de algoritmos de **Machine Learning**, contemplando desde etapas de análise e pré-processamento de dados até técnicas de regressão e classificação.
